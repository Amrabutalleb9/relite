# Relite

Premium software & marketing agency website. Built with Next.js, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Framer Motion**
- **TypeScript**

## Pages

- `/` — Home
- `/about` — About
- `/services` — Services
- `/work` — Work
- `/contact` — Contact
